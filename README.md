# PS2 Character Generator

A Blender add-on for generating stylized low-poly characters inspired by sixth-generation consoles.

## Features

- Procedural character generation
- PS2-era polygon budgets
- Anime/JRPG proportions
- Modular ears, tails, hair, and clothing
- Automatic materials
- Rigging support (planned)

## Requirements

- Blender 4.x

## Installation

1. Download the repository
2. Open Blender
3. Edit → Preferences → Add-ons
4. Install ZIP
5. Enable PS2 Character Generator
